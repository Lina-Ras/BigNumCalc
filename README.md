# BigNumCalc

 #### How to run:
<b>First option:</b>
  - download git repo
  - make sure that python 3.0 is installed and the library PyQt5 is present
  - run "python main.py"
    
<b>Second option:</b>
  - download main.exe from releases
  - run executable <br>
    Note: another black screen will appear with BigNumCalc :(
